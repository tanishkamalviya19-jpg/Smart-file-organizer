Title: File Organizer (Python)

 📌 Description
A Python-based tool that automatically organizes files in a selected folder into categories such as Images, Documents, Videos, and Others. It also includes a search feature to quickly locate files.

🚀 Features
- Automatically organizes files based on type
- Creates folders dynamically (Images, Documents, Videos, Others)
- Moves files into respective folders
- Search functionality to find files by name

🛠️ Technologies Used
- Python
- os module
- shutil module

▶️ How to Run
1. Clone the repository or download the code
2. Open terminal in the project folder
3. Run the script:
   python app.py
4. Enter the folder path when prompted

⚠️ Note
- Use a test folder to avoid moving important files accidentally
